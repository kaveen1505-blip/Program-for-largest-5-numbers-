GitHub is a website where developers store and manage their code and projects. It helps people save their work, track changes, and work together with others.

Why GitHub is Used 

To store code online
To work with team members
To track changes in code
To share projects with others
To learn from other developers

How GitHub Works 

First, create an account on GitHub
Create a repository (project folder)
Upload your code or files
Make changes and save versions
Share with others or work as a team

Example 

If you create a largest of 7 numbers program, you can:
Upload it to GitHub
Edit it anytime
Share the link with your teacher or friends

Advantages of GitHub 

Free to use
Easy to share code
Saves all versions
Good for teamwork
Helps in learning

Simple Explanation (Largest of 7 Numbers Program) 
First, we include stdio.h for input and output functions
Then we declare 7 variables to store numbers
The user enters 7 numbers
We assume the first number is largest
Then we compare it with the remaining numbers
If any number is greater, we update the largest value
Finally, we display the largest number
